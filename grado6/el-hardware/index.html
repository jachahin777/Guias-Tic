🖥️ Taller: El Hardware

Área: Tecnología e Informática

📘 Teoría para estudiar

1. ¿Qué es el hardware?

El hardware es el conjunto de partes físicas de un computador, es decir, todos los componentes que se pueden ver y tocar.

2. Explicación sencilla

Imagina que el computador es como una persona: el hardware sería el cuerpo (manos, ojos, cerebro físico) y el software serían las ideas o pensamientos.
“Si lo puedes tocar, es hardware.”

3. Ejemplos de hardware

  • ⌨️ Teclado → Sirve para escribir letras, números y comandos.
  • 🖱️ Mouse → Permite mover el cursor y hacer clic en la pantalla.
  • 🖥️ Monitor → Es la pantalla donde ves la información.
  • 🧠 CPU o Procesador → Es el “cerebro” del computador donde se procesan los datos.
  • 💾 Disco Duro → Es el almacenamiento principal en la computadora.
  • 🖨️ Impresora → Sirve para imprimir documentos en papel.
  • 🔊 Parlantes → Permiten escuchar sonido.
