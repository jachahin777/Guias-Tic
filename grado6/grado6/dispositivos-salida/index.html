📘 Repaso Teórico: El Hardware

¿Qué es? El hardware es el conjunto de partes físicas de un computador, es decir, todos los componentes que se pueden ver y tocar.

Analogía: Imagina que el computador es como una persona: el hardware sería el cuerpo (manos, ojos, cerebro físico) y el software serían las ideas o pensamientos. "Si lo puedes tocar, es hardware."

Ejemplos comunes:

  • ⌨️ Teclado → Escribe letras, números y comandos.
  • 🖱️ Mouse → Mueve el cursor y hace clic.
  • 🖥️ Monitor → Pantalla donde ves la información.
  • 🧠 CPU → Cerebro donde se procesan los datos.
  • 💾 Disco Duro → Almacenamiento principal.
  • 🖨️ Impresora → Imprime documentos en papel.
  • 🔊 Parlantes → Reproducen sonido.
