1. Software

El software está formado por todos los programas y datos almacenados en los dispositivos de almacenamiento. Incluye los sistemas operativos y las aplicaciones que permiten realizar diferentes tareas en el computador, como escribir textos, navegar por Internet, dibujar, hacer cálculos o reproducir videos.

2. Sistema Operativo

El sistema operativo es el programa más importante de un computador. Su función principal es permitir que el usuario pueda comunicarse con la máquina y utilizar sus recursos. También realiza tareas básicas como reconocer dispositivos periféricos, mostrar información en la pantalla, administrar archivos, carpetas, unidades de disco y memorias.

3. Windows

Windows es un sistema operativo desarrollado por la compañía Microsoft. Está compuesto por ventanas que representan tareas o programas en ejecución. Su función principal es servir como interfaz entre el usuario y la computadora, facilitando su uso.

Ejemplos: Windows 7, Windows 8, Windows 10, Windows 11, Windows Server, Windows Phone

Aplicaciones incluidas: WordPad, Bloc de notas, Paint 3D, Microsoft Edge, Cortana, Windows Store, Alarmas, reloj y calendario, Microsoft Defender, Calculadora

4. Linux

Linux es un sistema operativo gratuito. No tiene costo, se puede modificar y redistribuir con libertad. Es muy utilizado en servidores porque es estable, seguro y permite una gran personalización.

Distribuciones: Ubuntu, Mint, Debian, Red Hat, Fedora, Suse, CentOS

5. Android

Android es un sistema operativo utilizado principalmente en dispositivos móviles con pantalla táctil, como celulares y tabletas. También puede encontrarse en relojes inteligentes, automóviles, televisores, electrodomésticos y otras máquinas.

Versiones: Cupcake, Ice Cream Sandwich, Jelly Bean, KitKat

Aplicaciones incluidas: Play Store, Gmail, Calendario, Google Drive, Contactos, Google Maps, Micrófono, Traductor, Fotos, Meet, Play Libros
